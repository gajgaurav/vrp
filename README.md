# VRP

Personal learning repo for vehicle routing problems. Working through Toth & Vigo (2014) and implementing solvers on VRP-REP benchmark instances.

## Structure

- `parsers/` — instance file parsers (VRP-REP XML, TSPLIB format)
- `solvers/` — MILP formulations and metaheuristics
- `bks/` — best-known solutions for validation
- `instances/` — benchmark instances
- `notebooks/` — analysis and visualization

## References

- Toth, P. & Vigo, D. (2014). *Vehicle Routing: Problems, Methods, and Applications*, 2nd ed., MOS-SIAM.
- VRP-REP: http://www.vrp-rep.org
